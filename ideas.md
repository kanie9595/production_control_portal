## Концепция 1: "Industrial Command Center" (Индустриальный Командный Центр)

**Design Movement:** Industrial Brutalism meets Data Dashboard — вдохновлено интерфейсами промышленных SCADA-систем и военных командных центров, но с современной эстетикой.

**Core Principles:**
1. Информационная плотность — максимум полезных данных на экране без перегрузки
2. Функциональная строгость — каждый элемент служит цели, никаких декоративных излишеств
3. Контрастная иерархия — критические элементы мгновенно привлекают внимание
4. Тактильная отзывчивость — интерфейс реагирует на каждое действие

**Color Philosophy:** Тёмно-серая основа (#1a1d23) с акцентами цвета расплавленного металла (amber/orange #f59e0b) для активных элементов и холодного стального синего (#64748b) для фоновых. Зелёный (#22c55e) для "выполнено", красный (#ef4444) для "критично". Философия: цвет = статус, а не украшение.

**Layout Paradigm:** Левая боковая панель навигации (фиксированная, узкая) + основная рабочая область с карточной системой. Верхняя полоса статуса как в кабине пилота.

**Signature Elements:**
1. Прогресс-бары в стиле "заполнения резервуара" — вертикальные и горизонтальные
2. Статусные индикаторы-точки с пульсирующей анимацией для активных задач

**Interaction Philosophy:** Чёткие переключения состояний с микро-анимациями. Checkbox → плавное заполнение с лёгкой вибрацией цвета.

**Animation:** Плавные slide-in для панелей, мягкое свечение при hover, пульсация для незавершённых критических задач. Минимальная длительность — 150ms, максимальная — 300ms.

**Typography System:** Заголовки — JetBrains Mono (моноширинный, индустриальный характер), основной текст — Inter (читаемость), акценты — полужирный uppercase для категорий.

## Концепция 2: "Clean Shift Board" (Чистая Сменная Доска)

**Design Movement:** Swiss Design / International Typographic Style — вдохновлено швейцарской школой дизайна с её чистотой, сеткой и типографикой как главным инструментом.

**Core Principles:**
1. Типографическая иерархия — шрифт управляет вниманием, а не цвет
2. Математическая сетка — всё выровнено по строгой модульной сетке
3. Минимальный хроматизм — ограниченная палитра, цвет только для статусов
4. Белое пространство как структура — пустота создаёт группировку

**Color Philosophy:** Почти монохромная основа: белый (#fafafa), тёплый серый (#78716c), графитовый (#292524). Единственные цветовые акценты — функциональные: зелёный (#16a34a) для выполненного, янтарный (#d97706) для "в процессе", красный (#dc2626) для просроченного.

**Layout Paradigm:** Горизонтальные табы сверху (День/Неделя/Месяц) + вертикальный список задач с широкими отступами. Никаких карточек — только линии-разделители и типографика.

**Signature Elements:**
1. Крупные числовые счётчики прогресса (72px) в стиле табло аэропорта
2. Тонкие горизонтальные линии-разделители с точками-маркерами статуса

**Interaction Philosophy:** Минималистичные переходы. Checkbox — мгновенное зачёркивание текста с fade. Никаких "вау-эффектов", только точность.

**Animation:** Только функциональные: fade-in при появлении, strikethrough при завершении задачи. Длительность 200ms, easing: ease-out.

**Typography System:** Заголовки — Space Grotesk (геометрический гротеск), основной текст — IBM Plex Sans (нейтральный, читаемый), числа — Space Mono (табличные цифры).

## Концепция 3: "Factory Floor Dashboard" (Панель Управления Цехом)

**Design Movement:** Neo-Industrial с элементами Material Design 3 — современный промышленный интерфейс с мягкими формами, но индустриальным характером.

**Core Principles:**
1. Контекстная навигация — боковая панель с иконками секций (День/Неделя/Месяц/Рекомендации)
2. Карточная архитектура — каждая группа задач в отдельной карточке с собственным прогрессом
3. Цветовое кодирование по времени — утро/день/вечер имеют свои оттенки
4. Адаптивность — одинаково удобен на планшете в цеху и на десктопе в кабинете

**Color Philosophy:** Светлая основа с тёплым подтоном (#f8f6f4). Основной акцент — глубокий индиго (#3730a3) для навигации и заголовков. Утренние задачи — тёплый amber (#fbbf24), дневные — нейтральный slate (#475569), вечерние — cool blue (#3b82f6). Статусы: emerald (#059669) для выполненного, rose (#e11d48) для пропущенного.

**Layout Paradigm:** Фиксированная левая sidebar с иконками + основная область с горизонтальным скроллом карточек-групп. Dashboard-виджет прогресса сверху.

**Signature Elements:**
1. Круговые диаграммы прогресса (donut charts) для каждой секции
2. Цветные боковые полосы на карточках, обозначающие время суток/период

**Interaction Philosophy:** Плавные, но заметные переходы. Карточки слегка приподнимаются при hover. Checkbox с анимацией "галочки" и мягким цветовым переходом фона строки.

**Animation:** Spring-based анимации для карточек (framer-motion), staggered появление списков задач, плавное обновление прогресс-баров. Длительность 250-400ms.

**Typography System:** Заголовки — Plus Jakarta Sans (современный, дружелюбный), основной текст — DM Sans (чистый, нейтральный), числа/метрики — Outfit (современные пропорции).
